Why does this happen?

Performance bias can happen for a different reasons:

  • Not enough training samples in the low-performing data slices
  • Wrong labels in the training set in the low-performing data slices
  • Drift between your training set and the test set

How to correct this?

We strongly recommend that you inspect the incorrect samples in the detected low-performing data slices. This will allow you to find the right feature engineering or data augmentation strategies to avoid such biases.
